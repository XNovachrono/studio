# **App Name**: Uncoverly Hub

## Core Features:

- Login and Authentication: User authentication for students and teachers with role-based access control.
- Student Onboarding: Wizard-like onboarding flow for new students to capture profile details.
- Student Home Dashboard: Display of scheduled classes, notes (Notion links), and books (PDFs) in a card-based layout.
- Student List (Teachers): Display students awaiting group assignment with interests, availability, and plan type.
- Group Creation: Teachers can create student groups based on compatible plan types and automatically generate unique group IDs.
- Content Management: Content management system for teachers to upload class links, notes, and book chapters for each group.

## Style Guidelines:

- Dark mode color scheme with blue, gray, and black as primary colors.
- Primary color: Deep blue (#2962FF) to evoke trust and professionalism within the dark interface.
- Background color: Very dark gray (#121212) for a modern, unobtrusive dark mode experience.
- Accent color: Pale blue (#ADD8E6) as a subtle contrast that still harmonizes with the blues, highlighting key UI elements without disrupting the clean look.
- Body and headline font: 'Inter', a sans-serif typeface providing a clean and readable text experience, for body text.
- Card-based UI elements for displaying classes, notes, and books with clean, floating designs.
- Font for headings: 'Space Grotesk' sans-serif for short chunks of text
